# Magic-island

This program is a simple text-based story game where players can choose how the story continues. 
Created as a final assignment for the Basic Programming course, this project was inspired by the "Magic Island" lore of TXT.

## 🛠️ Technologies
- **Language:** C++
- **Compiler:** Dev-C++ / g++
- **Libraries Used:** `<iostream>`, `<string>`, `<vector>`, `<ctime>`, `<cstdlib>`, `<algorithm>`, `<limits>`, `<cstddef>`, `<unistd.h>`

## 🚀 How to Compile and Run

### Using Dev-C++
1. Open the `MagicIsland.cpp` file in Dev-C++.
2. Press the **F11** key on your keyboard, or click **Execute → Compile & Run**.

### Using Terminal / Command Line (g++)
1. Open a terminal and navigate to the directory where the file is saved.
2. Run the following command to compile:
   ```bash
   g++ -o MagicIsland MagicIsland.cpp
